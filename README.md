# Zenframe

Zenframe is a calm digital planning system for modern families.

It brings together shared calendars, tasks, routines, and meal planning into one simple interface designed to reduce digital noise and make family life easier to organize.

## Features

- Shared family calendar
- Tasks and routines
- Meal planning
- Clean Nordic design
- Works across phones, tablets, and shared displays
